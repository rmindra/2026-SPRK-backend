# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-02-15
### Added
- CRUD Ruangan (Rooms).
- CRUD Peminjaman (Bookings).
- Approval workflow status booking (Pending/Approved/Rejected/Cancelled).
- Validasi bentrok jadwal saat approve booking.
- Search, filter, dan sorting untuk booking.
- Swagger UI untuk eksplorasi API.
- Setup database dengan EF Core migrations.
